# webserverNode

- Basique http server
- Log ip from request to json file 
- display request list with html

 ## START
- #### Install npm module 
    ```
    npm install 
    ```
- #### Start server
    ``` 
    nodemon server.js 
    ```
*(install nodemon to global usage > sudo npm install nodemon)*

#### Go to :
http://localhost:42333/
